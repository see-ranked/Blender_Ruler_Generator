Blender Ruler Generator (Python Script)
Blender内で、長さや目盛の間隔を自由にカスタマイズできる「定規」を自動生成するPythonスクリプトです。

概要
このスクリプトを実行すると、指定したパラメータに基づいた本体（ベース板）、目盛線、数値ラベルが自動的に作成されます。大量の目盛線を効率的に扱うため、目盛部分は自動的に一つのオブジェクトに結合され、専用のコレクションにまとめられます。

機能
長さの自由設定: 定規の全長を任意に指定可能。
目盛の間隔設定: 小さな刻み（Tick）と数字が入る大きな刻み（Label）の間隔を個別に設定可能。
フォントのカスタマイズ: 外部フォントファイル（.ttf）の指定および文字サイズの変更が可能。
自動最適化: 生成される大量の目盛線を結合し、シーン内のオブジェクト数を削減して動作を軽量化します。
使用方法
1. スクリプトの準備
Blenderを起動し、画面上部のタブから [Scripting] ワークスペースを選択します。
テキストエディタにある [+ New] ボタンをクリックして、新しいテキストファイルを作成します。
提供されたPythonスクリプトをコピーし、エディタに貼り付けます。
2. パラメータの調整
コードの最下部にある create_ruler() 関数の引数を書き換えて、好みの設定に変更してください。

create_ruler(
    length=20.0,            # 定規の全長
    tick_interval=0.1,      # 小目盛の間隔（例: 0.1 = 10cmごと）
    label_interval=1.0,     # 数字が表示される間隔（例: 1.0 = 1mごと）
    font_path="C:/Windows/Fonts/arial.ttf", # フォントファイルのパス（※重要）
    font_size=0.3          # 文字の大きさ
)
⚠️ 注意：フォントパスについて
font_path に指定するパスは、お使いのOSに合わせて書き換えてください。

Windows: C:/Windows/Fonts/arial.ttf など
macOS: /Library/Fonts/Arial.ttf など
指定しない場合: None と設定すれば、Blenderのデフォルトフォントが使用されます。
3. 実行
エディタ右上にある [Run Script] (再生ボタンアイコン) をクリックします。
3Dビューポートに定規が生成されます。
生成物の構成
生成されたオブジェクトは、自動的に Ruler_Collection という名前のコレクションに格納されます。

Ruler_Body: 定規のベース部分（メッシュ）
Ruler_Ticks: すべての目盛線（結合済みメッシュ）
Label_X.XX: 各数値ラベル（テキストオブジェクト）
カスタマイズガイド
さらに調整したい場合は、関数内の以下の変数値を変更してください。

変数名	内容	デフォルト値
ruler_width	定規の幅（奥行き）	0.5
ruler_thickness	定規の厚み	0.05
tick_length_small	小さな目盛線の長さ	0.1
tick_length_large	数字付き目盛線の長さ	0.2
label_offset	文字と定規の隙間（Y方向）	-0.3

ライセンス
このスクリプトはオープンソースとして提供されており、自由に改変・利用いただけます。
